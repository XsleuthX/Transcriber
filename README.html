Transcriber Editor

  1. Open index.html via a local server (modules): python -m http.server 8080 → http://localhost:8080. When deployed on GitHub Pages, just visit your Page URL.
  2. Click Import Video and select your file.
  3. Click Import SRT/VTT to load captions generated by your whisper-cli script.
  4. Use the FPS selector (24/25/30/50/60). Timecode panel & timestamps display as HH:MM:SS:FF.
  5. Edit any line; click Export SRT to download standard SRT (HH:MM:SS,mmm).

Note: SRT/VTT standards store milliseconds, not frames. The UI shows frames for precision; exports remain standards‑compliant.
